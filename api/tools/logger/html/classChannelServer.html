Detailed Description

The channel server consists of the zone the character interacts with.

Each Each channel server may contain all zones or a subset of zones requiring the client to switch to a different channel server for certsin zone changes. This class will handle incoming channel server connections and create a ChannelConnection object to proxy (and log) the connection between the client and the target channel server.
