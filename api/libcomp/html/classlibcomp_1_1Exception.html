Detailed Description

Base exception class.

Any application specific exceptions should be derived from this class. If an exception is not derived from this class (e.g. the bad_alloc exception), it should be caught and either handled or thrown again as an application exception using this class or any of it's derived classes. When constructing an exception, use the EXCEPTION macro for automatic insertation of the file and line the exception occured on.
