Apply modifications from a zone partial to an instanced zone definition.

Unique IDs and NPCs/objects in the same spot that already exist on the definition will be replaced by the partial definition, including deletes. If the zone definition supplied for this function matches the pointer stored for the base definition, it will fail.

Parameters
zonePointer to a zone definition to modify
partialIDID of the zone partial definition to apply to the zone
Returns
true if the modifications were successful, false if they were not or if the supplied zone definition is not a copy of the original
