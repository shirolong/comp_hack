Requirements

Running the Server

If running the server under Linux, you should install packages for the following:

  • Qt 4.8.x or higher (QtCore, QtNetwork, QtScript, QtSql, QtXml)

  • OpenSSL 0.9.8 or higher

  • MySQL 5 (recommened if you are running a production server or the website)

Compiling the Source Code

  • Working C++ compiler (like GCC, MinGW, LLVM/Clang, Visual Studio 2010)

  • Qt 4.8.x or higher (QtCore, QtGui, QtNetwork, QtScript, QtSql, QtWebKit, QtXml)

  • CMake 2.8 or higher

  • OpenSSL 0.9.8 or higher

  • zlib (Qt uses this but it is also used directly in the code)

  • Doxygen (if you wish to build the API documentation)

  • Docbook XSL and xmlto (if you wish to build this guide and other documention)

  • Git (if you wish to checkout a copy of the source code from the repository instead of using a pre-packaged distribution)
